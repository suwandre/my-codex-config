# Issues

This directory contains issue-specific notes and resolutions.

## Format

Files follow the naming convention: `<issue-id>.md`

Each issue file is an append-only log with timestamped notes.

## Example

```markdown
# Issue #123

## Notes

### 2024-01-26 10:00:00

Initial investigation shows the problem is related to dependency versions.

### 2024-01-26 14:30:00

Fixed by updating to latest version. Need to document the breaking changes.

### 2024-01-27 09:00:00

Added migration guide in CHANGELOG.md. Issue resolved.
```
